## Slice 1: Pin Codex model + remove plugin-authored per-call model

**Validation:** critical
**Goal:** Goal 2 — a stale per-call model id can never reach the Codex API through plugin-authored
workflows.

The audit proved (spec §4) that server-level `-c model="gpt-5.5"` only applies when the caller omits
the per-call `model`; a per-call `model` still overrides. So the fix is two-sided: pin the server
config AND make every plugin-authored `codex`/`codex-reply` snippet omit `model`.

### Tests required

1. **Manifest pin** — `.claude-plugin/plugin.json` `mcpServers.codex.args` contains
   `model="gpt-5.5"`, `sandbox_mode="danger-full-access"`, and `approval_policy="never"`.
   (Integration; reads the real manifest file; no mocks — it is the artifact under test.)
2. **No plugin-authored per-call model** — none of the plugin-authored skill files that open codex
   threads contains a top-level `"model"` / `model:` directive for the codex MCP tool. The audited set
   is `skills/brainstorming/codex-pairing.md`, `skills/brainstorming/SKILL.md`,
   `skills/systematic-debugging/SKILL.md`, and `skills/writing-plans/SKILL.md` (the last only for
   codex-MCP snippets — its implementer frontmatter is excluded). The only allowed mention is hazard
   text. (Integration; reads real skill files.) **Codex round-1 critique found the original three-file
   scope missed `brainstorming/SKILL.md:88` and `systematic-debugging/SKILL.md:175` — this test now
   covers the full set.**
3. **Hazard text allowed** — the stale `gpt-5.2-codex` literal may appear only on a line that also
   says it must NOT be passed. (Guards against the regression of someone "documenting" it as usable.)
4. **Future-file guard** — the test enumerates skill files by globbing `skills/**/SKILL.md` plus

2. **GREEN — rotation setter:** add to sidecar.js, async + `withSidecarLock` (the rotation is a
   multi-field mutation; do it atomically):

   ```js
   export async function setCodexThreadId(
     specPath,
     { role = 'paired-reviewer', oldThreadId, newThreadId, reason, phase, round },
   ) {
     if (typeof newThreadId !== 'string' || newThreadId.length === 0) {
       throw new Error('setCodexThreadId: newThreadId must be a non-empty string');
     }
     return withSidecarLock(specPath, () => {
       const sc = loadSidecar(specPath);
       if (!sc.role_sessions || typeof sc.role_sessions !== 'object') sc.role_sessions = {};
       sc.role_sessions[role] = newThreadId;
       if (role === 'paired-reviewer') sc.codex_session = newThreadId;
       if (!Array.isArray(sc.thread_rotations)) sc.thread_rotations = [];
       sc.thread_rotations.push({
         old_thread_id: oldThreadId ?? null,
         new_thread_id: newThreadId,
         role,
         reason: reason ?? null,
         phase: phase ?? null,
         round: Number.isInteger(round) ? round : null,
         rotated_at: new Date().toISOString(),
       });
       saveSidecar(specPath, sc);
       return sc;
     });
   }
   ```

   ```js
       feature: sc.feature ?? null,
       artifact: specPath,
       goals: goalsBlock ?? null,
       rounds: (sc.rounds ?? []).map((r) => ({
         phase: r.phase, round: r.round, claude: r.claude, codex: r.codex,
       })),
       open_contentions: sc.open_contentions ?? [],
       thread_rotations: sc.thread_rotations ?? [],
     };
   }
   ```

   ```js
   export async function appendRoundWithAudits(specPath, { audits, round }) {
     if (!Array.isArray(audits)) throw new Error('appendRoundWithAudits: audits must be an array');
     if (!round || typeof round !== 'object') throw new Error('appendRoundWithAudits: round required');
     return withSidecarLock(specPath, () => {
       const sc = loadSidecar(specPath);
       if (!Array.isArray(sc.audits)) sc.audits = [];
       // 1. validate ALL audits against current schema (throws → nothing saved)
       const normalized = audits.map((a) => normalizeAuditEntry(a)); // throws on bad entry
       // 2. validate the round's SHIP claims against audits-in-scope (existing + incoming)
       const scope = sc.audits.concat(normalized);
       assertRoundShipBacked(round, scope); // throws → nothing saved
       // 3. mutate + save once
       for (const a of normalized) sc.audits.push(a);
       sc.rounds.push(round);
       saveSidecar(specPath, sc);
       return sc;
     });
   }
   ```
