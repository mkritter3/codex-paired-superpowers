Steps use checkbox (`- [ ]`) syntax.

**Architecture:** Nine sequential slices: project-config (1) → worktree primitives (2) → sidecar persistence (3) → reconciler module (4) → subagent definition files (5) → cherry-pick integration (6) → provenance hook (7) → autopilot SKILL.md + structural smoke (8) → empirical parallel smoke + release (9). Decisions live in SKILL.md prose; mechanics live in Node modules.

**Tech Stack:** Node.js 20+, zero npm deps. Bash for hook + smoke tests. JSON-only for `.codex-paired/project.json`. All git shell-outs via `node:child_process.execFileSync` with argv arrays (no shell, no injection risk).

  - `create(repoRoot, sliceId, sliceStartSha)` creates worktree at `<repo>/.git-worktrees/slice-<N>` branched as `<slice-id>-impl`. Returns absolute worktree path.
  - `create` halts `worktree-gitignore-missing` if `.git-worktrees/` not in `.gitignore`.
  - `create` halts `worktree-path-conflict` if path exists and is not a clean worktree of the same slice.
  - `create` halts `worktree-create-failed` on `git worktree add` nonzero exit (with stderr surfaced).
  - `bootstrap(repoRoot, worktreePath, symlinks)` — `symlinks` is `[{path, required}]`. For each entry: if source exists, symlink. If source missing AND `required:true`, halt `worktree-bootstrap-failed`. If source missing AND `required:false`, skip silently. Symlink target already exists and is the correct symlink → success (idempotent). Symlink target exists but is a regular file or wrong target → halt.
  - `verifyBootstrap(worktreePath, recordedSymlinks)` — for each recorded path: `lstat` is a symlink; `readlink` resolves to expected absolute path. Returns `{ok:true}` or `{ok:false, failed:[{symlink, reason, expected?, actual?}]}`. Reasons: `missing` / `not-a-symlink` / `wrong-target`.
  - `reset(worktreePath, sliceStartSha)` — `git reset --hard`. Preserves untracked symlinks. Halts `worktree-reset-failed` on nonzero exit.
  - `remove(repoRoot, worktreePath)` — `git worktree remove`. Halts `worktree-cleanup-failed` on nonzero exit.
  - `removeBranch(repoRoot, branchName)` — `git branch -D`. Halts `worktree-branch-cleanup-failed` on nonzero exit.
  - All halt returns `{ok:false, halt:{reason, detail}}`; success returns `{ok:true, ...payload}`.
